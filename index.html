Generative AI

App to generate structured Radiology reports from voice notes


  1. A radiologist can speak out their impressions while looking at X-rays and the app transcribes the audio in real time.
  2. The notes get converted into a well-structured radiology report using generative AI. The user can save, edit and delete reports.
  3. The user can also add macros to automatically replace technical terms in the audio transcription.
  4. Finally, the user can manage multiple report templates for different types of X-rays.
